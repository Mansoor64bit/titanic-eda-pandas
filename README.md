# Titanic Dataset EDA using Pandas

## Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python and Pandas.

The objective was to analyze passenger information, survival trends, and demographic patterns through data cleaning, transformation, and statistical analysis.

---

## Tools Used

- Python
- Pandas
- NumPy

---

## Analysis Performed

- Data Cleaning
- Missing Value Analysis
- Survival Analysis
- Passenger Class Analysis
- Age Analysis
- Gender Analysis
- Descriptive Statistics
- Data Aggregation and Grouping

---

## Dataset Features

The dataset contains information such as:

- Passenger ID
- Name
- Age
- Gender
- Passenger Class
- Fare
- Embarked Port
- Survival Status

---

## Key Insights

- Female passengers had higher survival rates.
- First-class passengers were more likely to survive.
- Age and socioeconomic status influenced survival probability.
- Missing values required cleaning before analysis.
